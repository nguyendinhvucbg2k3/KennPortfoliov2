=======================================
HƯỚNG DẪN CẬP NHẬT HOẠT ĐỘNG
=======================================

Toàn bộ các hoạt động của bạn được quản lý tại một nơi duy nhất để dễ dàng chỉnh sửa.

---------------------------------
1. Vị trí tệp tin
---------------------------------
Mở tệp: `src/lib/placeholder-data.ts`

Tìm đến một mảng (array) có tên là `experiences`. Mỗi hoạt động là một đối tượng (object) trong mảng này. Trang web sẽ tự động sắp xếp các hoạt động theo mốc thời gian khi hiển thị.

---------------------------------
2. Cách chỉnh sửa
---------------------------------
- **Để thêm hoạt động mới:** Sao chép một khối đối tượng `{...}` có sẵn, dán nó vào trong mảng và thay đổi nội dung cho phù hợp.
- **Để sửa hoặc xóa:** Tìm hoạt động bạn muốn chỉnh sửa và thay đổi nội dung bên trong, hoặc xóa toàn bộ khối đối tượng đó.

---------------------------------
3. Cấu trúc của một đối tượng Hoạt động
---------------------------------
Mỗi hoạt động cần có cấu trúc song ngữ (tiếng Việt và tiếng Anh) như sau. Mô tả nên ngắn gọn và súc tích.

```javascript
{
  id: "1", // ID duy nhất cho mỗi hoạt động, không được trùng nhau.
  title: {
    vi: "Vai trò/Chức vụ của bạn (tiếng Việt)",
    en: "Your Role/Position (in English)"
  },
  organization: {
    vi: "Tên câu lạc bộ/tổ chức (tiếng Việt)",
    en: "Name of club/organization (in English)"
  },
  date: "Thời gian tham gia (ví dụ: 10/2021 — nay)",
  description: {
    vi: "Mô tả ngắn gọn, súc tích về vai trò và trách nhiệm chính (tiếng Việt).",
    en: "A brief, concise description of your main role and responsibilities (in English)."
  },
},
```

**Ví dụ:**

```javascript
export const experiences: Experience[] = [
    {
      id: "1",
      title: {
        vi: "Graphics Designer",
        en: "Graphics Designer"
      },
      organization: {
        vi: "PHENIKAA ESPORTS CLUB",
        en: "PHENIKAA ESPORTS CLUB"
      },
      date: "10/2021 — nay",
      description: {
        vi: "Thiết kế và quản lý các sản phẩm truyền thông cho CLB, đảm bảo nhận diện thương hiệu cho các sự kiện và giải đấu.",
        en: "Designed and managed media products for the club, ensuring brand identity for events and tournaments."
      },
    },
    {
      id: "6",
      title: {
        vi: "CTV Vận hành sự kiện",
        en: "Event Operations Collaborator"
      },
      organization: {
        vi: "FLAZER THUỘC TEAM FLASH VIETNAM",
        en: "FLAZER (UNDER TEAM FLASH VIETNAM)"
      },
      date: "07/2024 — Nay",
      description: {
        vi: "Hỗ trợ vận hành các sự kiện và giải đấu, giám sát và đảm bảo quy trình diễn ra suôn sẻ, chuyên nghiệp.",
        en: "Assisted in operating events and tournaments, monitoring and ensuring a smooth and professional process."
      },
    }
    // Thêm các hoạt động khác của bạn ở đây...
];
```

**LƯU Ý QUAN TRỌNG:**
- Đảm bảo mỗi `id` là duy nhất.
- Sau khi chỉnh sửa và lưu tệp `placeholder-data.ts`, trang chủ sẽ tự động cập nhật lại dòng thời gian hoạt động của bạn với ngôn ngữ tương ứng.
- Tiêu đề của mục (ví dụ: "Hoạt động") được quản lý trong tệp `src/lib/content.ts` nếu bạn muốn thay đổi.
